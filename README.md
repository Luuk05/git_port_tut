## Css framework

Het framework dat gebruikt word is Bootstrap.

## One page template

Dit product is een one page template, die bedoelt is om een persoonlijke portfolio website te maken. Hierin staan dingen zoals de services die je levert, projecten die je hebt gemaakt/ mee bezig bent en een contact form. Modern en zeker niet rommelig. 

## Gebruik

Dit product kan gratis gebruikt worden. Je opent de html files en verandert de nodige informatie.
